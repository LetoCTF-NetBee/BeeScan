## Описание модулей:
### redirection.py
- проверка на переадресацию. 
- принимает: URL 
- возращает: T/F и настоящий URL
- пример: redirection.testRedirection('http://larri.ru')
- вернет: False, 'https://www.aviasales.ru/'
